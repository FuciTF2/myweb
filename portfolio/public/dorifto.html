Project

Dorifto

A drift racing game built in Python. My first proper project — rough around the edges, held together with hope and pygame, and somehow works.

Actual gameplay. Yes those are drifts.

What is it

Dorifto is a top-down drift racing game where you drive around a track, try not to fly off the edge, and score points by maintaining a drift angle. The longer and harder the drift, the higher the score. It has a HUD, an achievement system, NOS, a handbrake.

It's not polished. The collision detection has opinions. But it runs, it's fun, and I built it from scratch, so I'm proud of it.

Why it matters

This was my first proper project — the first time I sat down with no tutorial, no guided exercise, and just built something because I wanted to see if I could. Everything I've made since started here. It's where I figured out that I actually enjoy building things, not just learning about building things.

It's written entirely in Python using Pygame. One of the few times I've touched Python for something that isn't a script.
